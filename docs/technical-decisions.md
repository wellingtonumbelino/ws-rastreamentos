# Decisões Técnicas

## Escolha de JavaScript puro

O projeto foi implementado utilizando JavaScript puro para manter simplicidade e demonstrar domínio da linguagem sem dependência ou auxílio de _framework_.

## Estrutura de arquivos simples

A estrutura foi mantida minimalista para facilitar o entendimento e a leitura do código, mas também organizada de forma a garantir uma boa manutenibilidade.
